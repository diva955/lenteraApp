berikut langkah langkah install flutter

1. Persiapan Awal(install flutter)

Minimal yang dibutuhkan:

RAM: 8 GB (disarankan)
Storage: ±10–20 GB kosong
2. Download Flutter SDK
Buka website resmi:
👉 https://flutter.dev
Download Flutter SDK sesuai OS kamu
Ekstrak file zip:
Contoh (Windows):
C:\src\flutter
3. Set PATH (Penting!)

Agar bisa pakai perintah flutter di terminal:

Windows:
Buka Environment Variables
Edit Path

Tambahkan:

C:\src\flutter\bin

4. Cek Instalasi

Buka terminal / CMD lalu ketik:

flutter doctor

Nanti akan muncul checklist seperti:

Android toolchain
Chrome
VS Code / Android Studio

Kalau ada ❌, ikuti instruksi yang muncul.

5. Install Android Studio (untuk emulator)

Install:
👉 Android Studio

Saat install:

Centang:
Android SDK
Android SDK Platform
Android Virtual Device
6. Setup Emulator
Buka Android Studio
Masuk ke:
AVD Manager
Buat device baru:
Pilih: Pixel (bebas)
Pilih API terbaru
Klik Run emulator
